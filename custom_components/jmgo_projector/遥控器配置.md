# JMGO 投影仪遥控器配置指南

安装后，集成为每个遥控按键创建了独立的 **Button 实体**。

## 📱 创建的实体

重启 Home Assistant 后，会看到以下实体（实体名称取决于你设置的设备名称）：

### Button 实体（遥控器按键）
- `button.设备名称_power` - 电源
- `button.设备名称_ok` - 确认
- `button.设备名称_return` - 返回
- `button.设备名称_up` - 上
- `button.设备名称_down` - 下
- `button.设备名称_left` - 左
- `button.设备名称_right` - 右
- `button.设备名称_settings` - 设置
- `button.设备名称_menu` - 菜单
- `button.设备名称_option` - 选项

### Number 实体（音量控制）
- `number.设备名称_volume` - 音量（0-100，滑块控制）

### Media Player 实体
- `media_player.设备名称` - 媒体播放器（支持开关机、音量调节）

## 🎨 在仪表板中显示遥控器

### 音量控制

音量通过 `number.设备名称_volume` 实体控制，显示为 0-100 的滑块。

**Mushroom 音量卡片：**
```yaml
type: custom:mushroom-number-card
entity: number.客厅投影仪_volume
name: 音量
icon: mdi:volume-high
display_mode: slider
layout: horizontal
```

**标准卡片：**
```yaml
type: entities
entities:
  - entity: number.客厅投影仪_volume
    name: 音量
```

更多音量控制配置请查看 `lovelace-volume.yaml.example`。

### 方法 1：简单按钮排列（推荐新手）

1. 进入仪表板编辑模式
2. 点击 **+ 添加卡片**
3. 选择 **按钮 (Button)** 卡片
4. 添加所有 button 实体

或使用 **水平/垂直堆栈** 组合多个按钮。

### 方法 2：使用 Glance 卡片

1. 添加 **概览 (Glance)** 卡片
2. 配置：
   ```yaml
   type: glance
   title: 投影仪遥控
   columns: 5
   entities:
     - button.客厅投影仪_power
     - button.客厅投影仪_up
     - button.客厅投影仪_ok
     - button.客厅投影仪_down
     - button.客厅投影仪_return
   ```

### 方法 3：自定义布局（推荐）

参考 `lovelace.yaml.example` 文件中的完整配置。

复制配置到你的仪表板 YAML 编辑器中，**记得替换实体名称**为你的实际实体 ID。

## 🔧 获取实体 ID

1. 进入 **配置** -> **设备与服务** -> **实体**
2. 搜索 "jmgo" 或你的设备名称
3. 复制所有 button 实体的 ID

## 💡 高级配置

### 使用 Mushroom Cards（推荐，最美观）

Mushroom 是现代、美观的 Home Assistant 卡片库，提供多种卡片类型。

**安装：**
1. HACS -> 前端 -> 搜索 "Mushroom"
2. 安装并重启 Home Assistant
3. 在仪表板资源中添加：`/hacsfiles/lovelace-mushroom/mushroom.js`

**配置示例：**

```yaml
# 使用 Chips 卡片创建遥控器
type: vertical-stack
cards:
  - type: custom:mushroom-title-card
    title: 投影仪遥控

  # 方向键
  - type: custom:mushroom-chips-card
    alignment: center
    chips:
      - type: template
        entity: button.客厅投影仪_up
        icon: mdi:chevron-up
        icon_color: blue
        tap_action:
          action: call-service
          service: button.press
          target:
            entity_id: button.客厅投影仪_up
      - type: template
        entity: button.客厅投影仪_ok
        icon: mdi:checkbox-marked-circle
        icon_color: green
        tap_action:
          action: call-service
          service: button.press
          target:
            entity_id: button.客厅投影仪_ok
      - type: template
        entity: button.客厅投影仪_down
        icon: mdi:chevron-down
        icon_color: blue
        tap_action:
          action: call-service
          service: button.press
          target:
            entity_id: button.客厅投影仪_down
```

更多 Mushroom 配置示例请查看 `lovelace-mushroom.yaml.example`。

### 使用 custom:button-card（需要 HACS 安装）

```yaml
type: custom:button-card
entity: button.客厅投影仪_power
name: 电源
icon: mdi:power
color: red
tap_action:
  action: toggle
styles:
  card:
    - width: 100px
    - height: 100px
    - border-radius: 50%
  icon:
    - width: 50px
```

### 创建完整遥控器界面

使用 **Picture Elements** 卡片，配合背景图片创建逼真的遥控器：

```yaml
type: picture-elements
image: /local/remote-bg.png
elements:
  - type: state-icon
    entity: button.客厅投影仪_power
    style:
      top: 10%
      left: 50%
  # ... 其他按键
```

## 🎮 使用示例

### 自动化：模拟遥控器序列

```yaml
automation:
  # 打开投影仪并进入主界面
  - alias: "启动投影仪主界面"
    sequence:
      - service: button.press
        target:
          entity_id: button.客厅投影仪_power
      - delay: "00:00:05"
      - service: button.press
        target:
          entity_id: button.客厅投影仪_menu
      - delay: "00:00:02"
      - service: button.press
        target:
          entity_id: button.客厅投影仪_ok
```

### 脚本：创建遥控器宏

```yaml
script:
  projector_launch_netflix:
    alias: "启动 Netflix"
    sequence:
      - service: button.press
        target:
          entity_id: button.客厅投影仪_power
      - delay: "00:00:08"  # 等待开机
      - service: button.press
        target:
          entity_id: button.客厅投影仪_menu
      - delay: "00:00:01"
      - repeat:
          count: 3
          sequence:
            - service: button.press
              target:
                entity_id: button.客厅投影仪_right
            - delay: "00:00:01"
      - service: button.press
        target:
          entity_id: button.客厅投影仪_ok
```

## 📝 注意事项

1. **实体名称**：根据你的设备名称调整，如"客厅投影仪"会变成 `ke_ting_tou_ying_yi`
2. **延迟**：在自动化中添加适当延迟，等待投影仪响应
3. **状态**：按钮实体没有状态显示，仅用于触发操作
